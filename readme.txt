Anonymous members for BuddyPress and BuddyBoss

== Description ==

Allow your BuddyBoss & BuddyPress website's members to join groups anonymously.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.s
3. There are no settings yet.
